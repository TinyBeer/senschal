# Seneschal - 个人向工CLI具箱

希望将 `Senschal` 打造成一个个人向 `CLI` 工具箱，尽可能涵盖常用功能：

1. `ssh` 客户端
2. `docker` 环境部署器
3. 配置生成器
4. `todo` 管理
5. 图片转化器
6. `protobuf` 文件工具
7. ...
